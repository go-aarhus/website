# Go Aarhus Website

Welcome to the official repository for the **[Go Aarhus](https://www.meetup.com/go-aarhus/)** community website! 

Go Aarhus is the Gopher community in Denmark's tech capital. Whether you're a seasoned wizard or a brand-new curious gopher, this is the place for you. We talk concurrency, performance, web services, and all things Golang.

## Contributing

We love contributions! If you'd like to add your company to the "Companies Using Go" wall, propose a talk, or improve the site:

1. Fork the repository
2. Make your changes
3. Open a Pull Request

The team members and companies lists are easily editable in `src/data/organizers.ts` and `src/data/companies.ts`.

## Development

This site is built with React, TypeScript, Vite, and Tailwind CSS.

To run it locally:

```bash
# Install dependencies
pnpm install

# Start the dev server
pnpm run dev
```
